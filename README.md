**Overview**  
This project analyzes the Netflix Titles dataset (Kaggle) to uncover trends in global content production, genre preferences, release patterns, and platform growth. The analysis was done using Python (Pandas, Matplotlib).

**Key Questions**
- How many Movies vs TV Shows does Netflix host?
- Which countries produce the most Netflix content?
- What are the trends in release years?
- Which genres/categories are most common?
